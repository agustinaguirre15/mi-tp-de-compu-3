# Currículum Vitae Personal

Este proyecto es mi currículum vitae, creado para practicar y aplicar lo que apredí. El objetivo fue crear un curriculum vitae sobre mí, en el cual debia llevar a cabo los conocimientos aprendidos en clase.

## Características del Proyecto

-   **HTML5 Semántico**: El documento está construido con etiquetas de HTML5 como: `<header>`, `<main>`, `<section>`, etc. lo que lo hace más organizado, fácil de entender y armado de manera correcta.
-   **Estilos Externos**: Todo el diseño visual, desde los colores hasta las tipografías y el espaciado, se gestiono desde el archivo CSS como decia en la consigna
-   **Diseño Responsivo**: Utilicé Flexbox para organizar el contenido en columnas y apliqué una `media query` para asegurarme de que el diseño se adapte automáticamente a pantallas más pequeñas, como las de celulares.
-   **Tipografía Personalizada**: Para darle una identidad visual más propia, combiné dos fuentes: `Lora` para los títulos y subtitulos (que es una serif) y `PT Sans` para el texto principal (una sans serif). Esto crea una jerarquía clara y mejora la experiencia de lectura.
